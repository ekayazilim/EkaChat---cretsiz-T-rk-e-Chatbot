📚 Chatbot Kurulum Talimatları

1. Sistem Gereksinimleri

  • PHP 7.4 veya üzeri
  • MySQL 5.7 veya üzeri
  • Apache/Nginx web sunucusu
  • mod_rewrite modülü aktif olmalı

2. Dosyaların Yüklenmesi

  • Tüm dosyaları FTP veya File Manager ile sunucunuza yükleyin
  • Dosyaların izinlerini (chmod) ayarlayın: config.php: 644 yonetim/: 755 assets/: 755

3. Veritabanı Kurulumu

cPanel üzerinden:

  1. cPanel'de "MySQL Veritabanları"na tıklayın
  2. "Yeni Veritabanı Oluştur" bölümünden veritabanı adını girin
  3. "MySQL Kullanıcıları" bölümünden yeni kullanıcı oluşturun
  4. "Kullanıcıları Ekle" bölümünden kullanıcıya tüm izinleri verin

Plesk üzerinden:

  1. Plesk panelinde "Veritabanları"na tıklayın
  2. "Veritabanı Ekle" butonuna tıklayın
  3. Veritabanı adı ve kullanıcı bilgilerini girin

Veritabanı İçeriğinin Aktarılması:

  1. phpMyAdmin'e giriş yapın
  2. Oluşturduğunuz veritabanını seçin
  3. "İçe Aktar" sekmesine tıklayın
  4. "chatbot.sql" dosyasını seçin ve içe aktarın

4. Yapılandırma

config.php dosyasını düzenleyin:

// Veritabanı bağlantı bilgileri $db_host = "localhost"; // Genelde localhost $db_user = "veritabani_kullanicisi"; $db_pass = "veritabani_sifresi"; $db_name = "veritabani_adi";

5. Yönetici Paneli

Varsayılan giriş bilgileri:

Kullanıcı adı: admin Şifre: admin123
Önemli: İlk girişten sonra mutlaka şifrenizi değiştirin!

6. Yönetici Hesabı Oluşturma (Admin Silinirse)

Eğer admin hesabı silinirse veya giriş yapılamıyorsa:

  1. yonetim/add_admin.php dosyasını çalıştırın
  2. Yeni admin hesabı oluşturun
  3. İşlem bittikten sonra güvenlik için add_admin.php dosyasını silin

7. Chatbot'u Sitenize Ekleme

1. Önce chatbot.js dosyasını düzenleyin ve CHATBOT_URL değişkenini kendi sunucu adresinize göre güncelleyin:

// chatbot.js dosyasında bu satırı bulun ve değiştirin: const CHATBOT_URL = 'http://siteniz.com/chatbot';

2. Sitenizin footer kısmına şu kodu ekleyin:

<script src="http://siteniz.com/chatbot/chatbot.js"></script>
Önemli: URL'leri kendi domain adresinize göre değiştirmeyi unutmayın. Örneğin:
  • CHATBOT_URL = 'http://example.com/chatbot'
  • <script src="http://example.com/chatbot/chatbot.js"></script>
